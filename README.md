
# 健身俱乐部app

健身俱乐部app实现了视频的在线播放，调起百度地图，调起通讯录，调起邮箱，使用redis存储缓存数据，使用SQLite存储用户的个性设置数据，使用Rxjava+Retrofit实现了异步消息传输。
